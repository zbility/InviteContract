# InviteContract
Invite Contract 邀请合约

# Using
* 1、安装MetaMask准备好私钥信息；
* 2、拷贝邀请合约代码（Invite.sol）到 [Remix]（http://remix.ethereum.org） 或 [myetherwallet]（https://myetherwallet.com）；
* 3、填写合约测试信息：
  leader:   承销节点自己的地址
  truster:  调用合约地址
  decimal:  将奖励精确划分份额
  rule:     每层权重规则
* 4、确认无误后即可发布合约；
